#🚀 A 股多因子技术分析与报告生成工具

##📖 项目简介
这是一个基于 Python 的 A 股自动化筛选与报告生成工具。它利用 Akshare 库获取实时行情、研报、资金流向等多维度数据，并集成了多种技术分析指标（如双周期 MACD、KDJ、RSI 等）。程序通过多线程高效处理和整合信号，最终生成一份结构清晰、格式专业的 Excel 报告，帮助用户快速锁定具备潜在投资价值的股票。

##✨ 核心功能亮点

多线程高效率：采用 ThreadPoolExecutor 并行抓取历史数据和成分股，大幅提高数据获取速度。

双周期 MACD 策略：同时计算并提取 标准周期 (12, 26, 9) 和 加速周期 (6, 13, 5) 的 MACD 金叉信号。

全方位技术信号：支持 KDJ (超卖金叉), CCI (超卖), RSI (超卖低位), BOLL (低波/缩口) 等多种技术指标信号提取。

市场/基本面信号整合：整合主力研报买入评级、均线多头排列、强势股池、量价齐升及多日资金流向等信号。

智能数据管理：

自动进行数据清洗，过滤 ST/退市股票。

支持本地缓存机制，避免重复下载数据。

生成专业报告：将所有信号统一汇总，并生成包含多个工作表（如数据汇总、信号详情、资金流向等）的 Excel 报告。

##🛠️ 环境依赖与安装

本项目依赖 Python 3.6+ 环境。

您需要安装以下库：akshare, pandas, numpy, xlsxwriter, 和 pandas-ta。

pip install akshare pandas numpy openpyxl xlsxwriter pandas-ta

##🚀 使用方法

将提供的 Python 代码保存为文件，例如命名为 stock_analyzer.py。

在命令行或终端中运行该脚本：

python stock_analyzer.py

程序将自动执行以下步骤：

检查并获取所需数据（带有重试和缓存机制）。

筛选研报评级大于 1 的股票作为分析基础。

并行计算技术指标信号。

整合所有信号并过滤掉无有效信号的股票。

生成最终 Excel 报告 ~/Downloads/CoreNews_Reports/股票筛选报告_YYYYMMDD.xlsx

📋 代码架构概览核心功能封装在 StockAnalyzer 类中，主要方法包括

_get_all_raw_data()集中处理所有数据源的获取和缓存逻辑。

_fetch_hist_data_parallel()多线程获取股票历史数据。

_process_ta_signals()计算并提取所有技术指标信号（MACD/KDJ/RSI 等）。

_consolidate_data()核心方法：将所有数据源和信号合并、过滤，生成最终汇总表。

_generate_report()生成带格式的 Excel 报告。

##📖 技术分析指标 (TA Signals)

MACD_12269	

MACD（异同移动平均线）标准周期 (12, 26, 9)	金叉： 当前 DIF 线向上穿过 DEA 线。	传统的中期买入信号。DIF 上穿 DEA 表明短期动能开始超过长期动能，趋势可能由弱转强或加速。

MACD_6135	

MACD 加速周期 (6, 13, 5)	金叉： 当前 DIF 线向上穿过 DEA 线。	短期的、更灵敏的买入信号。参数更小，对价格变化反应更快，常用于捕捉短线快速拉升的启动点。

KDJ_Signal	

随机指标 (Stochastic Oscillator)	超卖区金叉： K 线向上穿过 D 线，且在最近 5 天内曾处于 超卖区域（K 或 D < 20）。	经过深度回调后，在低位区域（超卖区）形成向上突破。预示着短期反弹或反转的概率较高。

CCI_Signal	

顺势指标 (Commodity Channel Index)	超卖： 当前 CCI 值 低于 -100。	表明股价已严重偏离均值，处于极度超卖状态。被认为是强烈的反转信号，尤其在接近 -200 时。

RSI_Signal	

相对强弱指标 (Relative Strength Index)	超卖低位： 当前 RSI 值 低于 30。	表明股价在计算周期内下跌动能过强，市场处于非理性抛售状态，有技术性反弹的需求。

BOLL_Signal	

布林带 (Bollinger Bands)	低波/缩口： 最近 5 天的平均带宽（上下轨差距）低于历史平均带宽。	带宽缩小意味着股价波动率下降，进入盘整期。通常被认为是“暴风雨前的宁静”，预示着可能即将出现方向性的大幅波动。

均线突破数据	

实时最新价 > 10日均线 > 30日均线 > 60日均线。	最强烈的多头趋势信号。股价、短期、中期、长期均线依次向上排列，表明股票处于健康、持续的上升通道。

东方财富 "强势股池"	

股票出现在当日的官方强势股列表中。	意味着该股票在当日或近期表现出超越大盘的强劲势头，通常有资金和热点题材驱动。

同花顺 "量价齐升" 榜单	

股价上涨的同时伴随成交量放大。这是技术分析中最可靠的上涨信号之一，表明有新增资金入场追逐，支撑价格上涨。

同花顺 "连续上涨" 榜单	

股票连续上涨的交易天数。	反映了股价的持续性和稳定性。天数越多，表明短期趋势越强劲。

同花顺 "持续放量" 榜单	

股票持续放量（成交量明显放大）的交易天数。	反映了市场对该股票的关注度和活跃度持续上升，通常是启动行情的预兆。
